# React + TypeScript + Vite

#### How you implemented React and TypeScript features

I used React functional components with hooks like `useState` to manage dynamic data.

TypeScript interfaces helped me define clear structures for tasks, props, and form data.

I used controlled components for forms so React fully controls the input values.

Overall, I combined type safety with React logic to keep the code predictable and organized.

#### The challenges you encountered and how you overcame them

Understanding props and how data flows between parent and child components was confusing at first.

I also had some issues with TypeScript types not matching, especially with optional fields.

I fixed these by carefully aligning my interfaces and checking how props were passed.

Practicing small pieces step by step helped me get clarity and avoid errors.

#### Your approach to component composition and state management

I kept the main state in the parent component and passed data down using props.

Each component like TaskForm, TaskList, and TaskFilter has a specific responsibility.

I used functions in props to handle actions like updating or adding tasks.

This made the structure clean, reusable, and easier to manage as the app grows.
